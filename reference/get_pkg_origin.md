# Return string to display Origin depending on source of the assessment

Return string to display Origin depending on source of the assessment

## Usage

``` r
get_pkg_origin(source)
```

## Arguments

- source:

  return value of function `pkg_ref`

## Value

depending on source transformed string for the origin
