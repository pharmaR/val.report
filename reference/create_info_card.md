# Create a info card for the report

Create a info card for the report

## Usage

``` r
create_info_card(title, header = NULL, text = NULL, extra_class = NULL)
```

## Arguments

- title:

  text to include in the card title

- header:

  text or html content to be included in the card header

- text:

  content to be included as class card text

- extra_class:

  whether to add additional class for the card div in order to modify
  style

## Details

check https://getbootstrap.com/docs/5.0/components/card/ for class names
and card content To style the card, assign a custom class and style the
card within the quarto theme
